# TabBar Icons

This directory should contain the following icon files for the bottom navigation:

## Required Icons (32x32px recommended)

### Home Tab
- `home.png` - Inactive home icon
- `home-active.png` - Active home icon (brown color #8B4513)

### Category Tab  
- `category.png` - Inactive category/grid icon
- `category-active.png` - Active category icon (brown color #8B4513)

### Cart Tab
- `cart.png` - Inactive shopping cart icon
- `cart-active.png` - Active cart icon (brown color #8B4513)

### Profile Tab
- `profile.png` - Inactive user/profile icon  
- `profile-active.png` - Active profile icon (brown color #8B4513)

## Design Guidelines
- Use simple, clear iconography
- Maintain consistent visual style
- Ensure good contrast for accessibility
- Follow WeChat Mini Program icon guidelines
- Icons should be optimized for small sizes
